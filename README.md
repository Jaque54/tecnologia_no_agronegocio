# Igoragro2026 
Tecnologia no Campo - Projeto Agrinho 2026

Este projeto foi desenvolvido para o Programa Agrinho 2026 com o tema Tecnologia no Campo.

O objetivo do site é apresentar como a tecnologia está transformando o agronegócio por meio da inteligência artificial, sensores inteligentes, drones, Internet das Coisas (IoT) e agricultura de precisão.

Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

Conteúdo do projeto

- Tecnologias no Campo
- Inteligência Artificial no Agronegócio
- Produção Agrícola
- Benefícios da Tecnologia
- Expansão do Agronegócio
- Contato

Projeto Agrinho 2026.
